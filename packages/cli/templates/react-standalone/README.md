# {{title}}

CodeExpander 插件 - {{title}}（React 独立项目）

> 需要 `@codeexpander/dev-tools-ui`、`@codeexpander/dev-tools-i18n`、`@codeexpander/plugin-types` 已发布到 npm。若安装失败，请使用 `html` 或 `nodejs-stdio` 模板。

## 开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```

构建产物在 `dist/` 目录，可在 CodeExpander 中通过「导入目录」使用。
