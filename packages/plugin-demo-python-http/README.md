# plugin-demo-python-http

Demo plugin for a **Python HTTP backend** with CodeExpander.

## Runtime

- Backend executable: `python3` (resolved from system `PATH`)
- Uses **http mode**: long-lived process, listens on `CODEEXPANDER_BACKEND_PORT` and serves HTTP

----------

# plugin-demo-python-http（中文）

CodeExpander 插件 demo（Python）：**http 模式** 后端。

## 运行环境

- 后端可执行程序：`python3`（系统 `PATH` 中可找到）
- 使用 **http 模式**：常驻进程，从 `CODEEXPANDER_BACKEND_PORT` 读取端口并提供 HTTP 服务
