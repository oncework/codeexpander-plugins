# plugin-demo-php-http

Demo plugin for a **PHP HTTP backend** with CodeExpander.

## Runtime

- Backend executable: `php` (resolved from system `PATH`, using built-in web server)
- Uses **http mode**: long-lived process, listens on `CODEEXPANDER_BACKEND_PORT` and serves HTTP

----------

# plugin-demo-php-http（中文）

CodeExpander 插件 demo（PHP）：**http 模式** 后端。

## 运行环境

- 后端可执行程序：`php`（系统 `PATH` 中可找到，使用内置 Web 服务器）
- 使用 **http 模式**：常驻进程，从 `CODEEXPANDER_BACKEND_PORT` 读取端口并提供 HTTP 服务
