# Text Tools

Text utilities: word and character counter, text diff, case converter, regex tester, find & replace, add prefix/suffix, line break manager, remove duplicate/empty/extra lines, em dash replacer. Switch between tools via tabs.

## Tools

| id | EN |
|----|----|
| word-counter | Word/Character Counter |
| text-diff | Text Diff Checker |
| case-converter | Case Converter |
| regex-tester | Regex Tester |
| add-prefix-suffix | Add Prefix/Suffix to Lines |
| line-break-manager | Add/Remove Line Breaks |
| find-replace | Find and Replace Text |
| remove-duplicate-lines | Remove Duplicate Lines |
| remove-empty-lines | Remove Empty Lines |
| remove-extra-spaces | Remove Extra Spaces |
| em-dash-replacer | Em Dash Replacer |
